## What This Does

Anime Card Collect is a web automation script for browser-based card collection games with anime styling. It is meant to reduce the time spent on routine collection by applying speed changes and built-in auto-targeting (aimbot) behavior, helping players gather cards more quickly during normal gameplay or events.

Its main purpose is to make card selection and collection reactions faster and more consistent. This release includes the core automation needed for rapid collection, while future updates are intended to improve performance and broaden support for well-known anime card collection platforms.

## FAQ

**How do I install the script?**  
Download the file and then either paste it into the browser console or run it through a userscript manager.

**Will updates break the script?**  
Game updates can change behavior and affect how the script works. Check for script updates often.

**Can I customize the hotkeys?**  
Yes. Edit the options section in the script file to change the activation keys.

**Does it work with all card collection games?**  
It is aimed at general web-based anime card collection platforms, though some games may need small adjustments.

**Where are collected cards stored?**  
The script does not keep cards locally; all card data stays on the game server.
